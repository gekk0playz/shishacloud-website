# Shisha Cloud — Almada

**Premium shisha lounge website for Shisha Cloud, Almada, Portugal.**

Animated multi-page website with a smooth scroll experience, GSAP animations, custom cursor, and a WhatsApp reservation system.

## Live

Deployed on Vercel — auto-deploys on every push to `main`.

## Stack

- **HTML / CSS / JavaScript** — no frameworks, pure vanilla
- **GSAP 3 + ScrollTrigger** — scroll-driven animations and parallax
- **Lenis** — smooth scroll
- **Vercel** — hosting with clean URLs

## Pages

| Page | Path |
|------|------|
| Home | `/` |
| Menu | `/menu/` |
| Gallery | `/gallery/` |
| Reserve | `/reserve/` |

## Features

- Animated hero with aurora orbs, dot grid, and parallax background
- Full menu with real prices (shisha, cocktails, snacks, drinks)
- Photo gallery with lightbox and keyboard/swipe navigation
- Reservation form that sends a pre-filled WhatsApp message
- Page transitions and preloader animation
- Mobile-first responsive design

## Local Development

No build step needed — open `index.html` directly in a browser, or use any static file server:

```bash
npx serve .
```
