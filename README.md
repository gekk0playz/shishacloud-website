# Shisha Cloud — Almada

**Premium shisha lounge website for Shisha Cloud, Almada, Portugal.**

Animated multi-page website with smooth scroll, GSAP animations, custom cursor, and a WhatsApp reservation system.

## Live

Deployed on Vercel — auto-deploys on every push to `main`.

## Stack

- HTML, CSS, JavaScript — vanilla, no frameworks
- GSAP 3 + ScrollTrigger — scroll animations and parallax
- Lenis — smooth scroll
- Vercel — hosting

## Pages

- `/` — Home (animated hero, about, services, flavours, gallery, reserve CTA)
- `/menu/` — Full menu with real prices
- `/gallery/` — Photo gallery with lightbox
- `/reserve/` — WhatsApp reservation form

## Features

- Animated hero with aurora orbs and parallax background
- Real menu data: shisha from 15 EUR, cocktails, snacks, drinks
- Gallery lightbox with keyboard and swipe navigation
- Reservation form that opens a pre-filled WhatsApp message
- Page transitions and preloader animation
- Mobile-first responsive design

## Local Development

No build step. Open index.html in a browser or serve with:

```bash
npx serve .
```
